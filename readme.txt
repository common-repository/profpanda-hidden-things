Customize your Login screen, hide WooCommerce prices and buy button, change your Dashboard area and much more delicious things..

== Description ==
This Plugin allows you to customize your login screen by adding a logo, change the login's logo url to your site url, change the login's logo alternative text and change the Login Form colors. Allows you to deactivate your dashboards default itens and even works hiding woocommerce product's price and buy button (to show only for logged users) and redirect users to login page (at your choice) when they are not logged in and hit Add to Cart button. We have added an option to show the Midia Library images only for the user that upload it avoiding that anyone could delete images from other users.

== Installation ==
1. Upload the plugin files to the '/wp-content/plugins/profpanda_hidden_things' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->ProfPanda Hidden Things screen to configure the plugin
